# Day04：STL 基础

## 课后习题

以下仅列出习题的要求，具体的程序实现、分析过程及运行结果请参阅每个习题对应的文件夹。

### Q1：交换值

- 实现一个模板函数，接受两个参数。交换他们的值，并运行调试打印交换前后的值。

### Q2：通用计算器

- 创建一个通用的 Calculator 类模板，支持加、减、乘、除运算，并测试整型数据、浮点型数据的计算。

### Q3：班级人数比较

- 编写一个模板函数，实现比较两个班级的人数，包括总人数，男女生人数。
  - （1）求两个输入的最大值。
  - （2）支持自定义比较函数。

### Q4：成绩分析

- 输入 5 个学生的成绩（整数，范围 0~100），存储到 vector<int> 中，输出所有成绩的平均值（保留 2 位小数）并输出最高分和最低分。

### Q5：打印最大 k 个数

- 编写一个子函数，输入为一组整数，打印最大的 k 个数。

### Q6：英汉词典

- 利用 map.实现一个简单的英汉词典，要求输入中文或者英文，给出对应的翻译词汇，如果没有输出默认提示。
