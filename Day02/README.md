### Q3: 青蛙跳台阶算法问题与框图绘制

题目描述：一只青蛙一次可以跳跃一个台阶或两个台阶。输入台阶数 `n`，计算青蛙一共有多少种跳跃方法，结果存入变量 `m`。

- 初步实现：使用递归算法直接计算每个台阶的跳法，但此方法存在重复计算，效率较低。
- 优化方法：采用记忆化递归算法，通过缓存存储已计算的结果，避免重复计算，从而提高效率。
- 进一步优化：采用空间优化算法，仅保留最后两次计算的跳法结果，既保持了高效性，又降低了空间复杂度。

具体的框图可以在 Q3 的 README 文件中查看。

### Q4：反转字符串中的单词

题目描述：编写一个程序，接收用户输入的一句话，反转其中单词的顺序。

#### 程序功能

- **输入验证**：检查用户输入是否仅包含字母和空格。
- **单词反转**：将输入的句子按单词分割后反转单词的顺序并输出。
- **空格校验**：针对输入的句子中的空格数量做了专门处理确保反转的准确性。

### Q6:点和矩阵

题目描述：定义一个 Point 类表示平面上的一个点，再定义一个 Rectangle 表示平面上的矩形，用 Point 类的对象作为 Rectangle 成员描述平面上矩形的顶点坐标。要求类 Point 中有相应的成员函数可以读取点的坐标值，类 Rectangle 含有一个函效用于计算并输出矩形的顶点坐标及面积。在主函数中对类 Rectangle 进行测试。

#### 设计思路

1. **Point 类**：表示平面上的点，包含 x 和 y 坐标。
2. **Rectangle 类**：用两个 Point 对象表示矩形的左上角和右下角。
3. **用户输入**：输入两个点坐标，程序判断是否能构成有效矩形。
4. **计算面积**：根据两点坐标计算矩形的面积。
5. **输入验证**：验证输入格式和位置是否合法，并给出错误提示。

### 选做题：设计一个简单的计算器

编写程序设计一个简单的计算器，要求支持用户输入如下格式的表达式：
操作数 1 运算符 op 操作数 2
然后，计算并输出表达式的值。支持的运算符包括：加（`+`）、减（`-`）、乘（`*`）和除（`/`）。

- 本题解决的问题：
  - 支持错误处理，确保输入合法；
  - 支持小数计算；
  - 支持第一个操作数为负数的情况。
